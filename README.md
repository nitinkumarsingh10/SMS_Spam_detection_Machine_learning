# SMS_Spam_detection_Machine_learning
# 📩 SMS Spam Detection

## 🚀 Project Overview

This project classifies SMS messages as spam or ham (not spam) using Natural Language Processing (NLP) techniques.

## 📂 Dataset

* File: `spam.csv`
* Contains:

  * SMS text
  * Label (spam/ham)

## ⚙️ Technologies Used

* Python 🐍
* Pandas, NumPy
* Scikit-learn
* NLP (TF-IDF / CountVectorizer)

## 🔍 Workflow

1. Text preprocessing (stopwords removal, stemming)
2. Feature extraction using TF-IDF
3. Model training
4. Model evaluation

## 🤖 Models Used

* Naive Bayes
* Logistic Regression

## 📈 Results

* High accuracy in detecting spam messages
* Efficient text classification model

## 💡 Key Insights

* Certain keywords strongly indicate spam
* NLP techniques improve classification performance

## ▶️ How to Run

```bash
pip install -r requirements.txt
jupyter notebook "SMS Spam.ipynb"
```
